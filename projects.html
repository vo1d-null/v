🐍🐍Snake Game Pygame🐍🐍

🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍
Python Snake🐍 game using Pygame library.
Guide snake to eat mice, grow.
Control with arrow keys.
Score is snake length.
Collision with mouse increases length, places mouse randomly.
Collision with self/boundary ends game.
🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍🐍

Classes:

Mouse (movement/rendering),
Snake🐍 (movement/rendering),
Game (init/game loop/collision/score).
Functions for background, sound, game over screen.
